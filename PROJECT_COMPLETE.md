## 🚀 Quick Start (5 Steps)

### Step 1: Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with MongoDB URI and JWT secret
npm run dev
```

### Step 2: Frontend Setup
```bash
cd frontend
npm install
cp .env.example .env
npm start
```

### Step 3: Register First Account
- Browser: http://localhost:3000
- Click "Register here"
- Fill in: username, email, password
- Click Register

### Step 4: Register Second Account
- Incognito/New Window
- http://localhost:3000
- Register with different credentials

### Step 5: Chat!
- Select user from list
- Send message
- See real-time delivery! ✅

## 📈 API Endpoints

### Authentication (4 endpoints)
| Method | Endpoint | Auth | Purpose |
|--------|----------|------|---------|
| POST | /api/auth/register | No | Create account |
| POST | /api/auth/login | No | Login user |
| GET | /api/auth/me | Yes | Get current user |
| GET | /api/auth/users | Yes | Get all users |

### Messaging (3 endpoints)
| Method | Endpoint | Auth | Purpose |
|--------|----------|------|---------|
| GET | /api/messages/:receiverId | Yes | Get chat history |
| POST | /api/messages | Yes | Send message |
| PUT | /api/messages/:messageId/read | Yes | Mark as read |

---

## 🔄 Socket.io Events (8 Events)

### Client → Server (4 events)
- `user_online` - User comes online
- `send_message` - Send message to receiver
- `typing` - User is typing
- `stop_typing` - User stopped typing

### Server → Client (4 events)
- `users_online` - List of online users
- `receive_message` - New message received
- `user_typing` - Someone is typing
- `user_stopped_typing` - Someone stopped typing
